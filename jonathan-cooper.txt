      5. Use the git checkout -b 'firstname-lastname' command to create your own branch.
      6. Add and commit changes on your branch using "git add ." and "git commit -m"
      7. Push back to Github with git push -u origin branch-name
      8. Click on the Branch:master button, name-branch and then click the New pull request button
      9. After double checking, click Create Pull Request.
